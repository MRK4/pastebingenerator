# PastebinGenerator

PastebinGenerator allows you to generate as many random links as you want.

## Installation and usage

You'll need python3.
Just clone and use.

```bash
sudo git clone https://github.com/MRK4/pastebingenerator

cd pastebingenerator

python3 main.py
```
